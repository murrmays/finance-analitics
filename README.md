# Анализ финансов

![Status](https://img.shields.io/badge/status-learning--project-blue)
![Progress](https://img.shields.io/badge/progress-20%25-brightgreen)

Автоматизация учета личных финансов. Сервис позволяет пользователю загрузить CSV-файл, 
после чего автоматически парсит данные, категоризирует траты и возвращает статистику. 
Пользователь мгновенно получает понятный и красивый дашборд со своими расходами
без необходимости вбивать каждую покупку вручную.

![React](https://img.shields.io/badge/React-18.x-blue)
![Vite](https://img.shields.io/badge/Vite-3.x-646CFF)
![JavaScript](https://img.shields.io/badge/JavaScript-5.x-3178C6)

**Проект в разработке. В коде могут содержаться заглушки. Этот файл будет обновляться.**

- **Разработка кастомных алгоритмов:** Написание собственного парсера CSV-файлов с обработкой краевых случаев.
- **Проектирование интерфейсов на React:** Создание интерактивной Drag & Drop зоны и маппинг данных.
- **Продвинутая работа с Java (Spring Boot):** Использование Java Streams API для сложной группировки, фильтрации и агрегации финансовых данных.
- **Организация клиент-серверного взаимодействия:** Работа с `FormData`, передача файлов (`MultipartFile`, `Blob`) и настройка CORS.
- **Визуализация данных:** Отрисовка графиков и дашбордов на основе агрегированной статистики.
- **DevOps & Архитектура:** Разделение приложения на микросервисы (фронтенд + бэкенд) и полная контейнеризация проекта с помощью Docker и Docker Compose.

## Что уже можно посмотреть:

1. Интерактивную Drag & Drop зону для перетаскивания файлов.
2. Чтение и парсинг банковских выписок (CSV) прямо в браузере с помощью самописного алгоритма.
3. Извлечение заголовков файла и вывод превью загруженных данных.

## Features

- **Умный импорт:** Чтение выписок из любых банков благодаря возможности ручного связывания колонок. Бэкенд получает уже нормализованные данные.
- **Интерактивный дашборд:** Мгновенное построение красивых графиков распределения финансов по категориям.

## Tech Stack

- **Frontend:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/) + Vanilla JavaScript
- **Backend:** [Java 17+] + [Spring Boot 3] + [Java Streams API]
- **Визуализация:** [Chart.js](https://www.chartjs.org/) + [React Chartjs 2]
- **Деплой:** [Docker] + [Docker Compose]
